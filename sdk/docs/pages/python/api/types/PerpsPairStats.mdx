# PerpsPairStats

24-hour price and volume stats for one pair. Keys are camelCase (indexer wire shape).

## Definition

```python
class PerpsPairStats(TypedDict):
    pairId: str
    currentPrice: str | None
    price24HAgo: str | None
    volume24H: str
    priceChange24H: str | None
```

## Fields

**`pairId`** тАФ `str`.

**`currentPrice`** тАФ `str | None`. Last trade price. `None` if no trades in the lookback window.

**`price24HAgo`** тАФ `str | None`. Price 24h ago. `None` on insufficient history.

**`volume24H`** тАФ `str`. 24-hour USD volume. Defaults to `"0"` on no trades.

**`priceChange24H`** тАФ `str | None`. Absolute price change over 24h.
