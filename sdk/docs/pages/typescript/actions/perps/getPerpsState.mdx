# getPerpsState

Read global runtime state — last funding time, vault share supply, insurance fund balance, treasury.

:::warning[DEX currently disabled]
The Dango DEX is currently disabled. Calls described on this page will not execute on the live network until the DEX is enabled.
:::


## Signature

```ts
function getPerpsState(
  client: Client,
  parameters?: { height?: number },
): Promise<PerpsState>
```

## Example

```ts
import { createPublicClient, createTransport, testnet } from "@left-curve/sdk"

const client = createPublicClient({ chain: testnet, transport: createTransport() })
const state = await client.getPerpsState()
```

## Parameters

**`height`** — `number`, optional. Block height. Default `0` (latest).

## Returns

**`PerpsState`** — `{ lastFundingTime, vaultShareSupply, insuranceFund, treasury }`.
