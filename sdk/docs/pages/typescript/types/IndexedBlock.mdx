# IndexedBlock

A block as returned by the indexer ‚Äî header plus transactions and cron outcomes.

## Definition

```ts
type IndexedBlock = {
  blockHeight: number
  createdAt: string
  hash: string
  appHash: string
  cronsOutcomes: string   // JSON-encoded outcomes
  transactions: IndexedTransaction[]
}
```
