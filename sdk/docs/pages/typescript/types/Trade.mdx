# Trade

A spot trade fill.

## Definition

```ts
type Trade = {
  addr: Address
  quoteDenom: Denom
  baseDenom: Denom
  timeInForce: TimeInForceOptions
  direction: Directions
  blockHeight: number
  createdAt: string
  filledBase: string
  filledQuote: string
  refundBase: string
  refundQuote: string
  feeBase: string
  feeQuote: string
  clearingPrice: string
}
```

## Fields

**`addr`** — `Address`. The trader.

**`direction`** — `"bid"` or `"ask"` (`Directions`).

**`filledBase`**, **`filledQuote`** — amounts filled in base/quote.

**`refundBase`**, **`refundQuote`** — amounts refunded if the order over-funded.

**`feeBase`**, **`feeQuote`** — fees paid.

**`clearingPrice`** — the price the trade cleared at.
