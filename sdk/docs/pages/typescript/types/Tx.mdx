# Tx

A signed transaction. Holds messages, sender, gas, credential, and metadata.

## Definition

```ts
type Tx<Credential = Json, Metadata = Json> = {
  sender: Address
  msgs: Message[]
  gasLimit: number
  credential: Credential
  data: Metadata
}

type UnsignedTx = Pick<Tx, "sender" | "msgs">
```

## Fields

**`sender`** ‚Äî `Address`. The signing/paying account.

**`msgs`** ‚Äî `Message[]`. The list of messages to execute.

**`gasLimit`** ‚Äî `number`. Maximum gas the tx may consume.

**`credential`** ‚Äî `Json` (typically `Credential`). The signature bundle.

**`data`** ‚Äî `Json` (typically `{ chainId, userIndex, nonce }`). EIP-712 metadata signed with the tx.
