# AccountInfo

Account record stored by the account factory (no address attached).

## Definition

```ts
type AccountInfo = {
  readonly index: AccountIndex
  readonly owner: number
}
```

## Fields

**`index`** — `AccountIndex` (`number`).

**`owner`** — `number`. The owning user's index.
