## What every SDK gives you

- **Account model.** Create users and accounts, manage keys, mint signing sessions.
- **Transactions.** Compose, sign, simulate, and broadcast. Poll for inclusion.
- **State queries.** Balances, contract state, DEX pairs, perps positions.
- **Subscriptions.** Live block, trade, candle, and event streams over WebSocket.
- **Rate-limit awareness.** Documented per-language patterns for the 167 req/10s and 30 sub/WS limits.

## Where to go next

- New to Dango? Each language section opens with a five-minute "First call" guide.
- Migrating from Hyperliquid? Start with the [Python migration guide](/python/migration/hyperliquid).
